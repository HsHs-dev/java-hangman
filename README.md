# Hangman Game

This is a small Java Hangman game. Follow the steps below to run it.

## Requirements

* Java (JDK 8 or newer)

## Clone the repo

First clone the reop using

```bash
git clone https://github.com/HsHs-dev/java-hangman.git
```

then change directory into it

```bash
cd java-hangman
```

## How to Compile

After changing directory to the game directory, run

```bash
javac Hangman/*.java
```

This will create the `.class` files inside the `Hangamn` folder.

## How to Run

After compiling, run the game with:

```bash
java Hangman.Main
```
