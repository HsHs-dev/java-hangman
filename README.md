# Hangman Game

This is a small Java Hangman game. Follow the steps below to run it.

## Requirements

* Java (JDK 8 or newer)

## How to Compile

Open a terminal in the project folder and run:

```bash
javac Hangman/*.java
```

This will create the `.class` files.

## How to Run

After compiling, run the game with:

```bash
java Hangman.Main
```
